# 🚀 NEXT-GEN TRADING HUB PRO

यह एक एडवांस ट्रेडिंग डैशबोर्ड और बॉट है जो **Indian Market (NSE/BSE)** और **Crypto Markets** को लाइव ट्रैक करता है। इसमें आपकी पर्सनल 3-EMA और Price Action स्ट्रैटेजी इनबिल्ट है।

## ✨ मुख्य फीचर्स (Features)

- **Multi-Timeframe Analysis:** 5m, 15m, 1h, और 4h का डेटा एक साथ चेक करता है।
- **Custom Strategy:** 
  - EMA 9, 15, 200 Stack (Fast, Medium, Long Trend)
  - Price Action (HH/LL Structure)
  - Advanced Pullback Detection (Wick Rejection logic)
- **5 Categorized Watchlists:** Indices, Crypto, Sectors, Global, और Favorites।
- **Interactive Charts:** RSI और EMA के साथ TradingView जैसा प्रोफेशनल लुक।
- **Search & Filter:** वॉचलिस्ट में किसी भी स्टॉक को तुरंत ढूंढने के लिए।
- **Real-time Signals:** "Final Buy" और "Final Sell" अलर्ट्स कन्फर्मेशन के साथ।

## 📁 प्रोजेक्ट स्ट्रक्चर (Project Structure)

- `app.py`: मुख्य सर्वर और UI फाइल।
- `strategy.py`: आपके ट्रेडिंग सिग्नल्स और इंडिकेटर्स का असली दिमाग।
- `watchlist.py`: आपके सभी स्टॉक्स और इंडेक्स की लिस्ट।
- `requirements.txt`: प्रोजेक्ट को चलाने के लिए ज़रूरी लाइब्रेरीज़।
